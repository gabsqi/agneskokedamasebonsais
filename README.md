# 🌿 Agnes Kokedamas & Renascença Bonsai

Um catálogo digital estático e responsivo desenvolvido com a abordagem **Mobile-First** para uma floricultura artesanal. O projeto foi projetado com foco total em conversão e experiência do usuário (UX).

🔗 **[Acesse o catálogo em produção aqui](https://gabsqi.github.io/agneskokedamasebonsais/)**

---

## 🎯 Objetivo e Foco em Conversão

O **Agnes Kokedamas** resolve o problema de marcas artesanais que precisam expor seus produtos sem a complexidade de um e-commerce tradicional. 

A jornada de compra foi simplificada: o usuário navega pelo catálogo e, ao escolher uma peça, o botão "Tenho interesse" o direciona para o WhatsApp com uma **mensagem pré-preenchida** identificando o produto desejado. Isso reduz o atrito e otimiza a taxa de conversão.

### 📱 Abordagem Mobile-First
O desenvolvimento priorizou telas menores (smartphones), considerando que a maior parte do tráfego vindo de redes sociais como o Instagram consome conteúdos via dispositivos móveis. A interface se expande graciosamente para telas maiores usando *Media Queries*.

---

## 🛠️ Recursos Técnicos Aplicados

Para criar o layout limpo e performático, foram aplicadas boas práticas de desenvolvimento web moderno com HTML5 e CSS3 puro:

- **CSS Grid & Flexbox:** Utilizados em conjunto para estruturar o grid de produtos e o alinhamento dos componentes internos de forma flexível.
- **Pseudo-elementos (`::before`/`::after`):** Aplicados para inserção de detalhes visuais e decorativos diretamente via estilo, mantendo o HTML semântico e limpo.
- **Variáveis CSS:** Centralização de cores da paleta botânica e fontes, facilitando futuras manutenções visuais no tema.
- **Mensagem via URL Parameters:** Integração direta com a API de link do WhatsApp para automatizar o texto do pedido do cliente.
